current projects

dudas.js

Library for processing various types of patent data from government sources (e.g., the USPTO) and non-government sources (e.g., Google)

Planned features:

  • Parser/Streaming for USPTO XML data
  • Standard library for patent-related objects
  • Scraping/parsing for publicly available XML/HTML

about patent.io

patent.io is a collaboration with the common goal of providing open source solutions for manipulating the endless amount of data generated by patent offices around the world

getting involved

we're always looking for people who are interested in helping out. we are currently interested in both developers or legal professionals to help out with developments/guidance, respectively, but are open to anyone.

If interested, get in touch with us on github
